# 📱 Smart Multi-Lecturer Availability Display System

## 📌 Overview
This project provides a real-time system to check lecturer availability using a mobile app and IoT display device.

---

## ❗ Problem Statement
Students often waste time visiting faculty cabins during short breaks without knowing lecturer availability.

---

## 💡 Solution
Lecturers update their status via a mobile app → data stored in Firebase → ESP32 displays it on a screen near stairs.

---

## 🛠 Tech Stack
- Flutter (Mobile App)
- Firebase Realtime Database
- ESP32 (IoT Device)
- Arduino IDE

---

## ⚙️ System Architecture
Mobile App → Firebase → ESP32 → Display

---

## 🚀 Features
- Real-time status updates
- Multi-lecturer support
- Scrolling display
- Low-cost solution

---

## 📂 Project Structure
- mobile_app → Flutter app code
- esp32_code → ESP32 program
- firebase → Database structure

---

## 🔧 Setup Instructions

### 1. Firebase Setup
- Create Firebase project
- Enable Realtime Database
- Add JSON structure

### 2. ESP32 Setup
- Add WiFi credentials
- Upload code using Arduino IDE

### 3. Mobile App
- Run Flutter app
- Update lecturer status
